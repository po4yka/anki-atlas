# Specs Directory

The `specs/` directory mixes two kinds of documents:

- current Rust crate and surface specs
- older rewrite and migration design notes

For the current public runtime, start with:

- [specs/16-cli.md](specs/16-cli.md)
- [specs/17-api.md](specs/17-api.md)
- [specs/18-mcp.md](specs/18-mcp.md)
- [docs/ARCHITECTURE.md](docs/ARCHITECTURE.md)

Some older specs still contain Python-oriented source references because they were captured during the rewrite process. Treat those as historical design context unless they have been explicitly rewritten for the Rust workspace.
