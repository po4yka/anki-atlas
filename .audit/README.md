# Audit Reports

Per-crate convention-compliance audit reports generated by `scripts/audit_crate.sh`
and the `presets/project-audit.yml` Ralph loop.

## Report Format

Each `<crate-name>.md` file contains:

| Section | Content |
|---------|---------|
| Summary table | Per-check PASS/WARN/FAIL with counts |
| Clippy | Warning details or "Clean" |
| Convention drift | unwrap/expect locations, missing #[instrument], wildcard re-exports |
| Recommendations | Prioritized fix list |

## Severity Ratings

| Status | Meaning |
|--------|---------|
| **PASS** | Zero issues |
| **WARN** | Convention drift only (unwrap, instrument, wildcards) |
| **FAIL** | Clippy or format failures (CI would reject) |

## Running

```bash
# Single crate
./scripts/audit_crate.sh common

# Full workspace (via Ralph)
ralph run -c presets/project-audit.yml
```

## Files

- `CURRENT_CRATE.txt` - Transient file used by the Ralph loop (not committed)
- `<crate-name>.md` - Audit report (generated, in .gitignore)
- `README.md` - This file (tracked)
